# Android-Constraint_layout
Repository of an App that consists of android Constraint Layout
<br>
Features included:
<br>
<br />
*Applied constraints <br />
*Chains <br />
*Baseline & center constraints <br />
*Gone margin <br />
*Horizonal & vertical biasing <br />
*Match constraints(0dp) <br />
*Guidline <br />
*Barrier <br />
*Group <br />
*Aspect ratio <br />
